Minecraft Manager 2.6.0 includes a new UI layout and bugfixes
<h2>What's New?</h2>

<ul>

<li>
<b>New UI Layout</b><br>
Minecraft Manager's UI Layout has been redesigned, giving you an easier-to-understand interface that's quick to navigate.

<br>

<li>

Profiles have been renamed to Instances, which is a name that's easier to understand and more consistent with the rest of the modded Minecraft community.

Thanks for using Minecraft Manager, I really appreciate it.
